# Web Config Schema v2 — Granular Search/Extract Configuration

**Status:** Ready for implementation  
**Supersedes:** `.opencode/plans/2026-03-23-searxng-backend-design.md`

## Overview

Split the single `web.backend` config key into separate `web.search.backend` (+ `web.search.url`) and `web.extract.backend` keys. Add SearXNG as a search backend and "native" (pure HTTP/requests) as an extract backend. Migrate existing users automatically.

---

## New Config Schema

```yaml
# ~/.hermes/config.yaml
web:
  backend: "firecrawl"        # EXISTING key — still the fallback for both tools
  search:
    backend: "searxng"        # NEW — overrides web.backend for web_search
    url: "https://searx.example.com/search?q=%s&format=json"  # required for searxng
  extract:
    backend: "native"         # NEW — overrides web.backend for web_extract
```

**Precedence (both tools):**
`web.{tool}.backend` → `web.backend` → env-key detection (legacy fallback)

**Valid backends per tool:**
- `web_search`: `firecrawl`, `parallel`, `tavily`, `searxng`
- `web_extract`: `firecrawl`, `parallel`, `tavily`, `native`

**`%s` in search URL** is replaced with `urllib.parse.quote_plus(query)` at query time. Validation: if backend is `searxng` and no URL is configured, return a clear error JSON.

### 2. `tools/web_tools.py` — Backend selection split

Replace `_get_backend()` with two functions:

```python
_VALID_SEARCH_BACKENDS = {"parallel", "firecrawl", "tavily", "searxng"}
_VALID_EXTRACT_BACKENDS = {"parallel", "firecrawl", "tavily", "native"}

def _get_search_backend() -> str:
    web = _load_web_config()
    # 1. Tool-specific config
    specific = web.get("search", {}).get("backend", "").lower().strip()
    if specific in _VALID_SEARCH_BACKENDS:
        return specific
    # 2. Generic web.backend
    generic = web.get("backend", "").lower().strip()
    if generic in _VALID_SEARCH_BACKENDS:
        return generic
    # 3. Env fallback (legacy)
    if _has_env("SEARXNG_URL"):
        return "searxng"
    if _has_env("TAVILY_API_KEY") and not _has_env("FIRECRAWL_API_KEY") and not _has_env("PARALLEL_API_KEY"):
        return "tavily"
    if _has_env("PARALLEL_API_KEY") and not _has_env("FIRECRAWL_API_KEY"):
        return "parallel"
    return "firecrawl"

def _get_extract_backend() -> str:
    web = _load_web_config()
    # 1. Tool-specific config
    specific = web.get("extract", {}).get("backend", "").lower().strip()
    if specific in _VALID_EXTRACT_BACKENDS:
        return specific
    # 2. Generic web.backend
    generic = web.get("backend", "").lower().strip()
    if generic in _VALID_EXTRACT_BACKENDS:
        return generic
    # 3. Env fallback (legacy — same as search, minus searxng)
    if _has_env("TAVILY_API_KEY") and not _has_env("FIRECRAWL_API_KEY") and not _has_env("PARALLEL_API_KEY"):
        return "tavily"
    if _has_env("PARALLEL_API_KEY") and not _has_env("FIRECRAWL_API_KEY"):
        return "parallel"
    return "firecrawl"
```

Keep the old `_get_backend()` as an alias calling `_get_search_backend()` temporarily so nothing else breaks.

### 3. `tools/web_tools.py` — SearXNG search implementation

```python
def _searxng_search(query: str, limit: int = 5) -> str:
    """Search using a SearXNG instance."""
    web = _load_web_config()
    # URL: config takes precedence over env var
    url_template = (
        web.get("search", {}).get("url", "").strip()
        or os.getenv("SEARXNG_URL", "").strip()
    )
    if not url_template:
        return json.dumps({"success": False, "error": "SearXNG backend selected but no URL configured. Set web.search.url in config.yaml or SEARXNG_URL env var."})

    import urllib.parse
    search_url = url_template.replace("%s", urllib.parse.quote_plus(query))

    headers = {"Accept": "application/json"}
    api_key = os.getenv("SEARXNG_API_KEY", "").strip()
    if api_key:
        headers["Authorization"] = f"Bearer {api_key}"

    try:
        with httpx.Client(timeout=30.0) as client:
            resp = client.get(search_url, headers=headers)
            resp.raise_for_status()
            data = resp.json()
    except httpx.HTTPError as e:
        return json.dumps({"success": False, "error": f"SearXNG request failed: {e}"})
    except Exception as e:
        return json.dumps({"success": False, "error": f"SearXNG error: {e}"})

    results = data.get("results", [])[:limit]
    normalized = [
        {
            "title": r.get("title", ""),
            "url": r.get("url", ""),
            "description": r.get("content", ""),
            "position": i + 1,
        }
        for i, r in enumerate(results)
    ]
    return json.dumps({"success": True, "data": {"web": normalized}}, ensure_ascii=False)
```

Wire into `web_search_tool()`:
```python

    return _searxng_search(query, limit)
```

### 4. `tools/web_tools.py` — Native extract implementation

```python
async def _native_extract(urls: List[str]) -> List[Dict[str, Any]]:
    """Extract web pages using plain HTTP requests."""
    import requests
    try:
        import html_to_markdown
        _has_html_to_markdown = True
    except ImportError:
        _has_html_to_markdown = False

    results = []
    for url in urls:
        try:
            resp = requests.get(
                url,
                headers={
                    "Accept": "application/json, text/markdown;q=0.9, text/html;q=0.8",
                    "User-Agent": "Mozilla/5.0 (compatible; HermesAgent/1.0)",
                },
                timeout=30,
                allow_redirects=True,
            )
            resp.raise_for_status()
            ct = resp.headers.get("Content-Type", "").lower()

            if "application/json" in ct or "text/markdown" in ct:
                content = resp.text
            elif _has_html_to_markdown:
                content = html_to_markdown.convert(resp.text)
            else:
                # Fallback: strip tags crudely
                import re as _re
                content = _re.sub(r"<[^>]+>", "", resp.text)

            results.append({"url": url, "title": "", "content": content, "error": None})
        except Exception as e:
            results.append({"url": url, "title": "", "content": "", "error": str(e)})
    return results
```

Wire into `web_extract_tool()`:
```python

    results = await _native_extract(safe_urls)
```

### 5. `tools/web_tools.py` — Update `check_web_api_key()`

```python
def check_web_api_key() -> bool:
    """Check if any web backend is available."""
    web = _load_web_config()
    # Native extract never needs an API key
    if web.get("extract", {}).get("backend") == "native":
        return True
    # SearXNG needs a URL (no API key required)
    searxng_url = web.get("search", {}).get("url", "") or os.getenv("SEARXNG_URL", "")
    if searxng_url:
        return True
    return bool(
        os.getenv("PARALLEL_API_KEY")
        or os.getenv("FIRECRAWL_API_KEY")
        or os.getenv("FIRECRAWL_API_URL")
        or os.getenv("TAVILY_API_KEY")
    )
```

Also update `requires_env` in the registry calls to add `"SEARXNG_URL"`.

### 8. `hermes_cli/config.py` — v10→v11 migration block

Add **before** the generic `_set_nested` / missing fields loop (to avoid empty-string defaults clobbering migrated values):

```python
# ── Version 10 → 11: split web.backend into web.search.backend + web.extract.backend ──
if current_ver < 11:
    config = load_config()
    web = config.get("web", {})
    old_backend = web.get("backend", "").strip()
    changed = False

    if old_backend:
        search_cfg = web.setdefault("search", {})
        extract_cfg = web.setdefault("extract", {})

        # Copy to per-tool if not already set
        if not search_cfg.get("backend"):
            search_cfg["backend"] = old_backend
            changed = True
        if not extract_cfg.get("backend"):
            extract_cfg["backend"] = old_backend if old_backend != "searxng" else ""
            changed = True

        # If old backend was searxng, prompt for search URL
        if old_backend == "searxng" and not search_cfg.get("url"):
            if interactive and not quiet:
                print("\n  SearXNG backend detected. The new schema requires a full URL template.")
                print("  Example: https://searx.example.com/search?q=%s&format=json")
                url = input("  SearXNG search URL (with %s placeholder): ").strip()
                if url:
                    search_cfg["url"] = url
                    changed = True

    if changed:
        config["web"] = web
        save_config(config)
        if not quiet:
            print("  ✓ Migrated web.backend to web.search.backend / web.extract.backend")
```

## Migration Behavior Summary

| Scenario | Result |
|----------|--------|
| Existing user with `web.backend: firecrawl` | `web.search.backend: firecrawl` + `web.extract.backend: firecrawl` copied automatically |
| Existing user with `web.backend: tavily` | Both copied to `tavily` |
| Existing user with `web.backend: searxng` (old plan users) | `web.search.backend: searxng`, `web.extract.backend: ""` (falls back to generic), prompted for URL |
| New user with no web config | Empty defaults; `check_web_api_key()` falls back to env detection |
| User sets `web.extract.backend: native` manually | Extract works with no API key |

## Key Design Decisions

1. **Empty strings = unset** in DEFAULT_CONFIG for `web.*` keys, not `None`, to match the existing `_load_web_config().get("backend", "").lower().strip()` pattern
2. **`_get_backend()` kept** as alias calling `_get_search_backend()` to avoid breaking any callers outside `web_search_tool()` / `web_extract_tool()`
3. **SearXNG URL in config, not env** — `web.search.url` is the primary; `SEARXNG_URL` env var is preserved as fallback for backward compat
4. **Migration runs BEFORE `_set_nested`** generic loop — prevents empty-string defaults from overwriting the migrated values
5. **`native` backend in `check_web_api_key()`** returns True immediately (no key needed) so the tool is enabled without any API key when `web.extract.backend: native`
6. **`html-to-markdown` import is guarded** with try/except so existing setups don't break if the package isn't installed yet (with a degraded fallback)
